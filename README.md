# BumpFree - feature/pending-events

当前分支 (`feature/pending-events`) 主要为 BumpFree 引入了强大的**待定事件（Pending Events）协作系统**、**深度的日历 UI 优化（毛玻璃特效）**以及**增强的沉浸式个性化配置**。

## 🌟 新增核心特性

### 1. 待定事件协作系统 (Pending Events)
- **事件创建与展示**：允许 Room 内成员除了固定课表外，临时创建“待定事件”（如社团例会、聚餐等安排）。
- **智能渲染**：待定事件以专属的显式样式嵌入到公共日历中，直观显示时间占用预期，支持跨越多日的长程事件切割渲染。
- **配置分离**：这套逻辑被封装在了完全独立的 `public.room_events` 数据库表中。

### 2. 沉浸式个性化与设置中枢 (Room Settings)
将之前离散的日历属性设置全部统一收纳进全新的全局设置面板（`RoomSettingsDialog`）：
- **基础排版适配**：支持在本地调节日历的字体大小范围（`0.5rem - 1.2rem`）与动态字体颜色展示。
- **成员代表色管理**：高级管理员及本人可自由更改周视图中代表自己的课程块颜色标签，并同步至全平台。
- **沉浸式自定义背景**：
  - 支持直接在面板中上传本地图片作为专属日历背景，拥有最高优先级。
  - 支持本地无级调节全局**背景图片透明度**。
  - **日程透明度独立调节**：新增“导入课表”与“自定义事件”的两个独立透明度控制滑块，系统智能绑定透明度下限不能低于全局图片透明度，防止您的事件“隐形”在复杂的背景图里。

### 3. 次世代极简日历 UI (Glassmorphism)
告别生硬死板的线条，我们对 `react-big-calendar` 进行了巨量的 CSS 深度覆写与布局重构：
- **毛玻璃视效核心**：对顶部操作栏（Toolbar）、日期表头（Header）以及左侧时间轴部分（Time Axis）应用了 `backdrop-filter: blur(12px)` 的黑化磨砂玻璃滤镜效果。
- **消除视觉割裂**：
  - 彻底抹平组件的外边距（Padding），使上述玻璃面板严丝合缝地贴紧屏幕左、上、右侧，告别无意义的黑边。
  - 对 Webkit Native 的右侧/底部**滚动条**进行全局重写，连同占位的 spacer gap 一同替换为了平滑的深色背景玻璃隧道和悬浮透明滑块。
- **全局背景贯穿**：将自定义背景图容器层级提升并锁定，确保所有面板之下都是同一张连贯的大图。
- **呼吸感高光骨架**：将日历内部的横向与纵向分割线（网格线）的透明度降至极低的 `10%` 进行“幽灵化”弱化，以此强力突显右侧色彩鲜明的日程块焦点。
- **重新设计的导航栏**：取消上下午显示，替换为极简对齐的 `HH:mm`。在最上方的 Toolbar 动态插入了跟随当前视角的 `yyyy年 M月`，并且采用了强制绝对定位使日期切换按钮 (`<` `今天` `>`) 能够拥有无论如何都不会偏移的完美居中视效。
